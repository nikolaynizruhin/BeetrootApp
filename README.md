# Beetroot Portal
### Built with Laravel 5.5 + Vue 2.5
## Installation
```
git clone
composer install
npm install
npm run dev
php artisan migrate
php artisan db:seed
cp .env.example .env
php artisan key:generate
php artisan storage:link
php artisan passport:install
storage/public/avatars/default.png
storage/public/logos/default.png
```
